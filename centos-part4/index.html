CentOS的常用软件安装(个人推荐)

发表于 2019-03-28 | 分类于 linux |

上一篇我们配置了源并进行了更新，那么本篇就该进入一些常用环境的配置了。

首先开机，进入CentOS。

Office 套件

打开终端，键入：

sudo yum install libreoffice

执行即可。

NTFS 驱动

sudo yum install ntfs-3g

版本管理

sudo yum install git

压缩/解压支持

sudo yum install unzip p7zip unar

FTP 客户端

sudo yum install filezilla

NFS 支持

sudo yum install nfs-utils

kernel-devel(vbox等软件需要)

sudo yum install kernel-devel
