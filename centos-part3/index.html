CentOS的常用开发环境配置

发表于 2019-03-28 | 分类于 linux |

上一篇我们配置了源并进行了更新，那么本篇就该进入一些常用环境的配置了。

首先开机，进入CentOS。

C/C++ 编译器

GCC

打开终端，键入：

yum install gcc gcc-c++ compat-gcc-rr compat-gcc-44-c++

执行，就会安装C和C++编译器以及对gcc4.4的兼容。

make

打开终端，键入：

yum install make cmake gdb

执行即可。

Python

想只安装python的同学，请出门右转：http://www.infonook.org/centos7-python3-install/

按照步骤进行安装即可。

Anaconda

打开链接：https://www.anaconda.com/distribution/

选择合适版本进行下载。

下载完毕后，打开终端，切换至下载路径下，键入：bash ./Anaconda3-x.x.x-Linux-x86_64.sh

执行安装即可。
