GNOME桌面环境下一些有用的小东西

发表于 2019-01-31 | 分类于 linux |

好久没写博文了，其实也有很多需要记录的，慢慢整理中…(都9102年了，这个人还这么咕…)

也算是用了有一段时间的gnome了，不过相较于gnome3我更喜欢gnome-classic。

不过gnome作为桌面我虽然用着顺手，也是被一些“神奇”的设计所雷到，所以随便做个持续更新的笔记。

NO FUCK HOT CORNER！！！

这个方(狗)便(屎)的设计被无数用户所吐槽，至少在centos的论坛里，我们讨论过好几个帖子了。

然而经历了那么多版本更迭，这个玩意居然还顽强的占据着左上角，也是醉了。

可能在移动领域却是比较方便吧，不过在PC使用中，我简直是无法理解这个设计。

每次写东西/浏览文件时候，鼠标稍微碰到左上角，就触发了。

目前比较好的解决方法是安装tweak-tool，然后启用扩展插件(extensions)。

去下载这样一个扩展插件：https://extensions.gnome.org/extension/118/no-topleft-hot-corner/

安装之后在tweak-tool里打开它，享受吧。

自定义你的Applications项目

我这个人就是喜欢用简单粗暴的工具，不喜欢鼓捣。

有这么一个东西，叫alacarte。

安装之后就会多一个叫Main Menu的应用，打开它就能自由定义你的Applications了。
