CentOS的基础配置(源)

发表于 2019-03-28 | 分类于 linux |

上一篇文章我们讲述了如何把CentOS安装在硬盘上，所以这一篇就是开始使用了。

首先开机，进入CentOS。

添加源

打开终端

然后按键盘上的Win键然后直接键入terminal回车，打开终端。

EPEL

加入EPEL(https://fedoraproject.org/wiki/EPEL)源：

直接在终端中键入：sudo yum install epel-release 即可。

ELRepo

加入ELRepo(http://elrepo.org/tiki/tiki-index.php)源：

打开上面的链接，按照官方指导进行安装即可：

首先，键入：rpm –import https://www.elrepo.org/RPM-GPG-KEY-elrepo.org

然后请按照最新版本键入：yum install https://www.elrepo.org/elrepo-release-7.x-x.el7.elrepo.noarch.rpm

即可。

Nux Dextop

加入Nux Dextop(http://li.nux.ro/download/nux/dextop/)源：

首先打开http://li.nux.ro/download/nux/dextop/el7/x86_64/ 这个链接，然后按下 Ctrl+F 进行关键字 nux-dextop 的查找，比对并找到最新版本。

切到终端，按照刚才找到的版本键入：sudo rpm -Uvh http://li.nux.ro/download/nux/dextop/el7/x86_64/nux-dextop-release-0-x.el7.nux.noarch.rpm

即可。

准备更新

yum-axelget

安装yum-axelget插件，用以加速(并行)yum的下载速度。

在终端中键入：sudo yum install yum-axelget

更新

在终端中键入：sudo yum update

执行更新。
